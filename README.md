# Fractals

## General Overview
Implementing several shape classes in Java and using a drawing class to create some graphics, including a fractal based on user entered shapes.

## Usage

> javac Circle.java 

> javac Rectangle.java 

> javac Triangle.java

> javac Driver.java 

> java Driver
